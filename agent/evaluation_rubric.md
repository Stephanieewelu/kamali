# Evaluation Rubric

Score each category 1-5.

## Customer impact
- 1: Unclear value
- 3: Partial value delivered
- 5: Clear, validated outcomes

## Reliability
- 1: No validation/rollback
- 3: Basic checks
- 5: Full checks with rollback and monitoring

## Communication
- 1: Sparse updates
- 3: Regular updates but missing details
- 5: Clear, structured, and proactive communication

## Safety & compliance
- 1: Security risk
- 3: Some controls
- 5: Full adherence to security controls

## Operational readiness
- 1: No runbooks or handoff
- 3: Partial runbooks
- 5: Complete runbooks, alerts, and on-call readiness
