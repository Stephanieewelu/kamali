# Playbook: Integration Delivery

## Objective
Deliver a working integration in the customer environment with validated success criteria.

## Steps
1. **Discovery**
   - Collect goals, success criteria, timeline, and constraints.
   - Confirm data sources, systems, and owners.
2. **Plan**
   - Define milestones, validation checks, and rollback steps.
   - Identify dependencies, access needs, and approvals.
3. **Build & Configure**
   - Implement integration steps in staging first.
   - Capture configuration changes and versioned artifacts.
4. **Validate**
   - Run functional checks aligned to success criteria.
   - Document results with timestamps.
5. **Handoff**
   - Provide runbook, monitoring guidance, and escalation path.
   - Confirm operational ownership.

## Deliverables
- Integration plan with milestones
- Validation checklist and results
- Deployment notes and rollback path
- Operational handoff summary
