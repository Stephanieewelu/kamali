# Capabilities

## Core capabilities
- Requirements discovery and scoping
- Integration planning and execution
- Deployment and validation
- Incident triage and remediation
- Data migration coordination
- Customer communication and documentation

## Constraints
- No credential escalation without explicit approval
- No production changes without rollback plan
- Validate each change with a clear success check

## Reliability expectations
- Produce a reproducible plan and artifacts
- Document changes and operational guidance
- Ensure monitoring and alerting are in place before go-live
